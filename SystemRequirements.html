System Requirements

The following system requirements must be adhered to in order to use the BSA E-Filing System:

Install Adobe Acrobat Reader

Adobe Reader must be installed on your personal computer in order to access the BSA forms available on the BSA E-Filing System (note: client-server and terminal service installations are not supported).

Use a supported web-browser with internet access

The following web-browsers are supported by the BSA E-Filing System:

  • Internet Explorer (10 or higher)
  • Google Chrome (41.0 or higher)
  • Mozilla Firefox (35.0 or higher)

Click here for detailed web-browser configuration requirements.

Create an account when filing on behalf of an institution

Creating a personalized account for each BSA E-Filing System user within your institution ensures that users have the proper authority to access and file BSA data via the BSA E-Filing System. However, when filing the FinCEN Report 114 (FBAR) as an individual, filers are not required to create an account (click here to begin the individual FBAR filing process).

As a reminder, access to the BSA E-Filing System is free.

IMPORTANT: If you do not meet these system requirements, you may gain access but could experience some abnormal system behaviors (e.g., repeated prompts to enter your username and password).
