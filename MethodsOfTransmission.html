Supported Methods of Transmission

The following methods of BSA data transmission methods are supported by the BSA E-Filing System. There are no direct fees associated with the BSA E-Filing System; however, indirect costs may be incurred by the filer depending on which method of transmission used:

Discrete (single form transmission)

A filer completes and submits a single BSA "discrete" form accessible via the BSA E-Filing System. Transmission of BSA data using this method is free and most suitable for lower volume filers.

When filing on behalf of an institution using this method, filers must enroll on behalf of their institution in order to file; however, when filing the FinCEN Report 114 (FBAR) as an individual, filers are not required to enroll (click here to begin the individual FBAR filing process).

Batch (multiple forms transmission)

A filer creates and submits a single XML "batch" file containing one or more BSA forms (of the same type) in accordance with the associated form type's XML schema and XML User Guide.

Transmission of BSA data using this method is free and most suitable for higher volume filers; however, indirect costs may be incurred by the filer when establishing a method/process of XML batch file creation, which is not a service provided by the BSA E-Filing System.

Filers may transmit a batch file manually via the BSA E-Filing System website interface, or automate the transmission process via secure direct-to-server connection. For more information regarding batch transmission automation, please click here.
